# firmware-decrypt
un-sign and decrypt DJI firmware (.sig files)

</br>

## Usage

```
$ ./dji_fwdec -d -t {firmware_filename}
$ ./dji_fwdec -u -i {sig_filename}
```

## Options

- -d: decompress
- -u: unsign
- -t: tarfile
- -i: sigfile
- -v: verbosity level. 최대 3(-vvv)
